# 📑 To Do List
Projeto de uma lista de tarefas com o objetivo de treinar e aperfeiçoar mais a utilização do TypeScript com o React.

## 💻 Tecnologias
- ReactJS
- CSS modules
- TypeScript
- React Icons
- UUID

## ⚙️ Funcionalidades
- Adicionar uma tarefa a lista.
- Editar uma tarefa da lista.
- Remover uma tarefa da lista.
